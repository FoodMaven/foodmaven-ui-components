<fm-characteristics-input> allows the user to select any number of characteristics as long as all the characteristics are compatible.

@Input() allCharacteristics:
{
     implicitCharacteristic: Characteristic[],
     scaledCharacteristic: Characteristic[],
     singleGroupCharacteristic: Characteristic[],
     multiGroupCharacteristic: Characteristic[]
}
This component also allows for the prepopulation of characteristics. The identified characteristics must exist in the list of all characteristics in order to be automatically populated.

@Input() identifiedCharacteristics: IdentifiedCharacteristic[]

where an identified characteristic is in the shape of:

{
     externalCompId: string,
     name: string,
     value: string,
     type: string
}
